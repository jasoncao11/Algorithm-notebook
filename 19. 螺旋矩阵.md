**给你一个 m 行 n 列的矩阵 matrix ，请按照顺时针螺旋顺序 ，返回矩阵中的所有元素。**

![algo8](./images/alg8.jpg)

```
输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[1,2,3,6,9,8,7,4,5]
```

```
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        res = []
        while matrix:
            #取出第一行
            res+=matrix.pop(0)
            #剩余矩阵逆时针翻转
            matrix = list(zip(*matrix))[::-1]
        return res
```
